# 图书管理系统 v1.0

一个功能完整的Python图书管理系统，包含**三种不同实现方式**，支持数据持久化。

## 核心功能

| 功能 | 说明 |
|------|------|
| 添加图书 | 录入书名、作者、ISBN、分类 |
| 查看所有 | 显示全部图书及状态 |
| 搜索图书 | 按书名/作者/ISBN模糊搜索 |
| 借阅图书 | 记录借阅人和借阅时间 |
| 归还图书 | 自动清除借阅记录 |
| 查看已借出 | 显示所有外借图书 |
| 图书分类 | 支持编程/文学/历史等分类 |
| ISBN验证 | 自动验证ISBN-10/ISBN-13格式 |
| 实时统计 | 显示总藏书/可借阅/已借出数量 |
| 数据持久化 | JSON文件存储，重启不丢失 |

## 三种版本

| 版本 | 技术栈 | 运行方式 |
|------|--------|----------|
| 控制台版 | Python | `python 控制台版/main.py` |
| Web版 | Flask + HTML/CSS | `python Web版/app.py` → http://localhost:5000 |
| 🖥GUI版 | Tkinter | `python 图形界面版/gui_main.py` |


核心功能
模块	功能
图书管理	添加、编辑、下架、搜索图书
读者管理	添加、编辑、注销、搜索读者，借阅历史，缴纳罚款
借阅管理	借书、还书、续借，查看当前借阅列表
统计报表	实时统计、分类统计、图书/读者排行榜
系统管理	数据备份、恢复备份

三种版本
